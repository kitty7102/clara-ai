# Clara AI Search Engine

Clara is an AI-powered search and refinement engine that improves LLM accuracy using real-time web data.

It is optimized for Korean language understanding with multilingual support.

---

## 🚀 Features
- Real-time web search  
- Content extraction & cleaning  
- RAG-based answer generation  

---

## 🌐 Live Demo
👉 https://clara.cndkorea.net

---

## 🧠 Why Clara?

Traditional LLMs often hallucinate or use outdated data.

Clara solves this by:

- Fetching real-time information  
- Cleaning and ranking content  
- Feeding structured data into LLM
